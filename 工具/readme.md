文件夹内文件说明：
=====

1. 实录-*.txt：

	* **作用**：各本原文，供 `编译用代码/数据/*.typ` 锚定用。防止编辑代码时，意外改动原文。
	* 是 古籍图片 → 计算机文字的录入。

3. 从typ代码还原原文.html：

	* **作用**：将 `编译用代码/数据/*.typ` 的代码，解析成实录，用于与本文件夹内的实录比较。
	* 在上方文本框粘贴 `*.typ` 的内容。
	* 下方文本框会实时显示还原的原文。
	* 使用方法：将 html 文件下载至本地，用浏览器打开。用 EmEditor 比较两种文本。

4. 编辑、编译PDF指南.md

	* **作用**：简短介绍  Typst 编译 PDF 的方法。
    * 介绍了 本地或在线编译方式，文档项目的简短介绍。
