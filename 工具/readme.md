文件夹内文件说明：
=====

1. 实录-*.txt：

	* **作用**：是各本的原文，供 `编译用代码/数据/*.typ` 锚定用。防止编辑代码时，改动了原文。
	* 是从 古籍图片 的录入。

3. 从typst代码提取实录.html：

	* **作用**：提取 `编译用代码/数据/*.typ` 的实录部分，用于与本文件夹内的实录比较。
 	* 使用方法：
  		1. 将 html 文件下载至本地，用浏览器打开。 
		2. 在网页 上方文本框 粘贴 `*.typ` 的内容。
		3. 网页 下方文本框 会实时显示提取的实录。
		4. 用 EmEditor 等工具 比较 提取的实录 与 本文件夹内的实录。

4. 编辑、编译PDF指南.md

	* **作用**：介绍 Typst 编译 PDF 的方法。
    * 简短介绍了 本地或在线编译方式、文档项目结构。
